# Mdv

[![MIT LICENSE](http://img.shields.io/badge/license-MIT-blue.svg?style=flat-square)](LICENSE)

`mdv` is a simple markdown viewer.

## Usage

### View

```
$ mdv path/to/file
```

### Edit

```
$ mdv -e path/to/file
```

## Installation

Run the following command:

```
$ go get -u github.com/masa0x80/mdv/...
```
